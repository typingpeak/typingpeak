# TypingPeak — Project Overview

## 1. Product Name
TypingPeak

---

## 2. Product Vision

TypingPeak is a public competitive typing platform designed to provide accurate, transparent, and hybrid typing performance measurement.

It combines:
- Standard WPM (full-duration scoring)
- Live WPM (actual time-used scoring)
- Real word count
- Character tracking
- Mistake tracking
- Accuracy percentage
- Structured typing lessons

The goal is to build a clean, professional typing experience that is both competitive and educational.

---

## 3. Mission

To provide a free, fast, and reliable typing platform that:

- Uses industry-standard calculation formulas
- Clearly explains scoring to users
- Includes structured typing lessons for improvement
- Supports accessibility (font resize, clean layout)
- Works smoothly on desktop and mobile
- Monetizes responsibly via Google AdSense

---

## 4. Product Type

Public Competitive Typing Platform  
Monetized through Google AdSense  
No login required (initial version)

---

## 5. Core Modules

TypingPeak consists of two main systems:

### A) Typing Test System
- Real-time typing engine
- Hybrid WPM scoring (Live + Standard)
- Accuracy calculation
- Mistake tracking
- Duration options (30s / 60s / 120s)
- Random categorized paragraph system
- Font resize accessibility
- Best score storage (local)
- Score explanation section

### B) Lessons System
- Beginner typing lessons
- Home row practice
- Top row practice
- Bottom row practice
- Common word drills
- Accuracy-focused drills
- Speed improvement drills
- Structured progression system

Lessons serve both:
- Skill development
- SEO content expansion

C) Career Productivity Module (Phase 3)

Description:

Career typing assessment

Productivity projection

Industry WPM benchmarks

Transparent time-saved calculation

Professional motivational framing

Keep tone aligned with:
Professional, Transparent, Performance-first.

## 6. Target Audience

TypingPeak is built for:

- Students preparing for exams
- Government job aspirants
- Job seekers (data entry roles)
- Competitive typists
- Developers
- Freelancers
- Office professionals
- Beginners learning touch typing

---

## 7. Core Differentiator

Unlike most typing websites, TypingPeak:

- Shows both Live WPM and Standard WPM
- Clearly explains scoring methodology
- Includes structured typing lessons
- Uses hybrid competitive measurement
- Avoids misleading inflated burst-only scoring
- Prioritizes clean UX over clutter
- Is transparent about formulas
- Is optimized for AdSense monetization without harming usability

## 9. Monetization Model

Primary Monetization:
Google AdSense

Ad placement strategy:
- Top banner (outside typing area)
- Sidebar (desktop only)
- Bottom content area
- Between lesson sections
- Never inside typing paragraph

User experience remains priority.
