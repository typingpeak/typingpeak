# TypingPeak — UX Guidelines

---

# 1. UX Philosophy

TypingPeak must feel:

- Clean
- Competitive
- Professional
- Transparent
- Fast
- Distraction-free

The UI should never feel cluttered or gamified excessively.
Performance clarity is the priority.

---

# 2. Visual Identity

Theme Style:
Minimal Dark Interface (default)

Design Principles:
- High contrast text
- Clear typography
- No unnecessary animations
- Focus on typing area
- Balanced spacing
- Responsive layout

---

# 3. Layout Structure

Each page should follow consistent structure:

Header (Navbar)
Main Content Area
Optional Sidebar (desktop only)
Footer (legal + links)

The typing box must always remain visually central.

---

# 4. Typing Test Page UX Rules

---

## A. Focus Priority

The typing paragraph is the main focus.

Rules:

- No ads inside paragraph
- No floating overlays near input
- Clear visual hierarchy
- Font resize available
- Cursor indicator visible

---

## B. Color Meaning

Green → Correct character  
Red → Incorrect character  
Gray → Untyped  
White → Active cursor  

Color meanings must remain consistent.

---

## C. Stats Display

Stats must show clearly:

- Standard WPM
- Live WPM
- Accuracy
- Mistakes
- Words
- Characters
- Best WPM

Stats must:
- Be readable
- Be aligned
- Not distract from typing
- Update smoothly

---

## D. Font Resize

Accessibility feature:

- A- decreases font size
- A+ increases font size
- Minimum: readable
- Maximum: not layout breaking

Font resize must:
- Only affect typing paragraph
- Not break layout
- Work on mobile

---

# 5. Lessons Page UX Rules

Lessons must feel:

- Educational
- Structured
- Calm
- Beginner-friendly

Lessons should:

- Include instructional text
- Include practice drill
- Avoid competitive pressure
- Avoid leaderboard logic
- Use structured headings

Ad placement must:
- Be outside practice typing area
- Not interrupt learning flow

---

# 6. Homepage UX Rules

Homepage must:

- Clearly explain product
- Include strong headline
- Include Start Test CTA
- Explain WPM and accuracy
- Include FAQ section
- Be SEO optimized

Homepage must not:
- Look empty
- Be overly minimal
- Be ad-heavy

---

# 7. AdSense UX Rules

Ads must:

- Be clearly separated
- Not mimic buttons
- Not interfere with typing
- Not shift layout during typing

Allowed placements:

- Top banner
- Sidebar (desktop)
- Bottom content area
- Between lesson sections

Never:

- Inside typing paragraph
- Over input area
- Blocking restart button

---

# 8. Mobile Experience

Mobile must:

- Keep typing area readable
- Keep stats readable
- Support font resize
- Keep duration buttons accessible
- Prevent layout overflow

Typing must feel smooth on mobile keyboard.

---

# 9. Interaction Rules

- Timer starts on first key press
- Restart instantly resets state
- No delayed transitions
- Minimal animation
- No unnecessary popups

"Test Completed" should feel clean, not flashy.

---

# 10. Future UX Expansion

Future enhancements may include:

- Progress bar
- Animated stats reveal
- Streak indicator
- Level system
- Dark / Light theme toggle
- Sound effects toggle

These must remain subtle and not distract from core typing experience.

---

# 11. UX Non-Negotiables

- No clutter
- No score inflation tricks
- No hidden formulas
- No intrusive ads
- No lag during typing
- No misleading UI

TypingPeak must always feel reliable and professional.

---

UX is the visible personality of TypingPeak.

It must reflect clarity, precision, and performance.
